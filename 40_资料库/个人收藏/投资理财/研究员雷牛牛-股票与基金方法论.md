# 研究员雷牛牛：股票与基金方法论

## 结论先行

研究员雷牛牛真正值得保存的，不是“跌 4% 就买”这个数字，而是一条完整的投资链路：

```text
先确认风险预算
-> 区分核心仓位和卫星仓位
-> 判断标的质量与估值
-> 分批买入并保留现金
-> 持续检查基金报告或公司基本面
-> 因估值、仓位或投资逻辑变化而减仓
```

其中，基金报告阅读、仓位纪律、避免追涨和减少情绪化交易具有较高的长期价值；“4%定投法能稳定提高收益”没有被可靠证明。它本质上是一个**按价格新低触发的网格买入规则**，不是估值模型，也不是传统的定期定额。

我的总体评价：

- **投教价值：高。** 能把复杂概念讲成普通人可执行的步骤。
- **行为管理价值：较高。** 预先写规则，能减少追涨、满仓和临场冲动。
- **选基和读报告框架：中高。** 关注经理风格、持仓、规模和定期报告，比看短期排行榜更扎实。
- **4%参数本身：证据不足。** 固定阈值忽略了不同资产的波动差异，也可能长期留有大量闲置现金。
- **直接照搬到个股：风险高。** 好基金或宽基有分散和成分更新机制，坏公司的股价则可能永久下跌。

## 方法论，用正常话解释

### 1. 先决定自己能承受多少，而不是先找“牛股”

先把生活备用金、短期会使用的钱和长期可承受波动的钱分开。投资仓位要由目标期限和可承受损失决定，而不是由当天看多或看空决定。

一个实用判断是：如果当前权益资产再跌一大段，是否会影响工作、睡眠或迫使自己在低位卖出。答案如果是“会”，通常说明仓位已超过实际风险承受能力。

### 2. 把资产分成核心仓位和卫星仓位

- **核心仓位**：目标是取得长期、分散的市场回报，通常更适合宽基、低成本和透明规则的工具。重点是长期持有和再平衡，不是频繁猜涨跌。
- **卫星仓位**：用于行业基金、主动基金或个股观点。收益弹性更高，判断错误的代价也更大，因此需要更小的风险额度和更严格的失效条件。

“核心”和“卫星”是风险职责，不是产品名称。红利、行业基金或知名经理管理的基金并不会因为被叫作核心就自动低风险。

### 3. 先判断买的是什么，再判断买得贵不贵

对基金，至少要看：

- 底层资产和业绩比较基准是什么。
- 主动基金经理任职多久，投资风格是否稳定。
- 规模、股票仓位、行业集中度和前十大持仓如何变化。
- 收益来自经理能力、风格行情，还是少数重仓股碰巧上涨。
- 管理费、销售服务费、交易成本和申赎限制。

对股票，先回答：

```text
公司靠什么赚钱？
利润来自哪个行业位置？
客户为什么持续选择它？
竞争者为什么难以复制？
利润是否能变成自由现金流？
管理层是否合理使用股东资金？
当前价格已经计入了多乐观的未来？
```

### 4. 低估不是“跌得多”

作者重视低估和分批买入，这个方向有价值，但必须把价格下跌和价值下降分开：

- 价格跌、盈利和竞争优势没变，可能是赔率改善。
- 价格跌、盈利恶化得更快，估值反而可能更贵，这叫价值陷阱。
- 行业指数有成分更新机制；单家公司可能永久失去竞争力，两者不能用同一套“越跌越买”。

估值百分位只能说明“相对自身历史贵不贵”，不能证明未来一定上涨，也不能跨行业直接比较。

### 5. 4%定投法到底怎么做

公开复盘中记录的完整规则是：

1. 把计划投资的总金额平均分为 10 份。
2. 以最近一次阶段新低为参照，净值再创新低且累计下跌约 4%时投入 1 份。
3. 上涨但没有创新低时不买，所以它是“只在新低买”，不是每跌 4%都买。
4. 当净值高于平均成本约 15%后进入止盈观察；随后从高点回撤约 4%时卖出，再开始下一轮。

用一句话说：**先预留十次资金，只在更低的价格分批买，盈利一段后用回撤触发卖出。**

### 6. 买完以后，靠报告验证而不是靠信仰

基金投后可以按季度或半年检查：

- 规模变化是净值上涨造成的，还是大量申购造成的。
- 经理说的策略与实际持仓是否一致。
- 股票仓位、行业和重仓股是否发生重要变化。
- 集中度、换手率和风格漂移是否超出最初预期。
- 经理、公司治理、费率或产品规则是否变化。

个股则要更新收入、利润、现金流、竞争格局、资本开支和最初投资假设。价格下跌本身不是加仓理由，原始逻辑仍成立才是前提。

### 7. 卖出的理由应高于“赚了多少”

更稳健的卖出顺序是：

1. 原投资逻辑被事实推翻。
2. 仓位因上涨超过风险上限，需要再平衡。
3. 估值已把过度乐观预期计入价格。
4. 资金目标或时间期限发生变化。
5. 最后才是机械止盈规则。

固定“盈利 15%后回撤 4%卖出”简单，但可能过早卖掉长期赢家，也可能在基本面已坏时等不到盈利。因此它更像交易纪律，不应替代基本面和资产配置判断。

## 对 4%定投法的校验

### 有效的部分

- 把临场判断改成预先承诺的规则，减少追涨和情绪交易。
- 强迫投资者保留现金，避免第一次下跌就满仓。
- 对高波动、长期逻辑仍成立且可能均值回归的分散资产，能形成有纪律的左侧买入。
- 操作简单，容易让新手理解“仓位比猜底更重要”。

### 主要缺陷

1. **4%不是通用尺度。** 对债券基金可能过大，对高波动行业或个股可能只是日常噪声。
2. **可能长期不在场。** 只有创新低才买，在持续上涨或先跌后回升的市场中会留下大量现金。
3. **10份资金不等于足够抗跌。** “优秀基金极端只跌约 40%”不是可靠约束，行业基金和股票完全可能跌得更多。
4. **价格规则不能识别价值陷阱。** 公司或基金风格已经恶化时，继续摊低成本会放大永久损失。
5. **退出规则过于机械。** 15%盈利和4%回撤没有同时考虑估值、税费、申赎费、资产波动和持有目标。
6. **它不是传统定投。** 权威定义中的定期定额是在固定时间投入固定金额，不因市场涨跌停止；4%法属于条件触发买入。

### 第三方回测给出的信号

[2022年的独立复盘](https://finance.sina.cn/fund/jjgdxw/2022-05-27/detail-imizmscu3672651.d.html?vt=4)以 148 只成立满 5 年、近 5 年年化收益较高且基金公司规模靠前的基金，回测 2018-2021 年：

- 2018-2019 年的先跌后涨阶段，触发买入的平均间隔约 2 个月，投入本金偏少。
- 2020-2021 年上涨阶段，即使把触发幅度降到 3%，148 只基金中也只有 28 只触发买入，说明策略容易错过上涨。
- 一个医疗基金案例中，4%策略收益率约 13.8%，比普通月定投高约 3.8 个百分点，但总收益金额接近，普通定投更简单。

这份回测也有局限：样本按回测时点已知的“近 5 年高收益基金”筛选，存在幸存者偏差和后视偏差；它不能证明普通定投在所有时期必胜。但它足以反驳“只在新低买一定更优”，并明确暴露了闲置现金和错过趋势的问题。

## 我的改良建议

不把 4%法当主系统，而把其中有用的纪律拆出来：

```text
基础层：按目标期限确定股、债、现金的风险预算
核心层：对分散、低成本的长期资产执行固定周期投入
增强层：当估值和资产配置偏离达到预设条件时，使用预留资金加仓
个股层：每次加仓前重新验证基本面，设置单一公司和单一行业的风险上限
退出层：优先按逻辑失效和再平衡处理，机械止盈只做辅助
复盘层：记录决策时的信息、假设和结果，避免事后改写理由
```

对普通人而言，**固定定投保证长期在场，条件加仓利用明显波动**，通常比把全部资金都交给“创新低 4%”更完整。不同资产应根据波动率、估值和风险预算设计阈值，不应共用一个百分比。

## 一页决策清单

买任何股票或基金前回答：

| 问题 | 最低要求 |
|---|---|
| 这笔钱何时会用？ | 确认不是短期刚性支出和应急资金 |
| 底层资产是什么？ | 能说清主要持仓、行业、地区和风险来源 |
| 为什么可能赚钱？ | 写出盈利或指数增长的因果链，不只写“跌很多” |
| 为什么现在不算贵？ | 给出估值指标、比较基准和适用边界 |
| 最多买多少？ | 预先定义总仓位和分批计划 |
| 什么事实证明我错了？ | 至少一条可观察的失效条件 |
| 如何持续跟踪？ | 明确季度报告、财报、持仓和费用检查项 |
| 何时卖？ | 逻辑失效、再平衡、估值或资金目标，顺序清楚 |

## 名词解释

- **定期定额（DCA）**：固定时间投入固定金额，不因为市场涨跌临时停止。
- **净值（NAV）**：基金资产扣除负债后，分摊到每份基金的价值。
- **回撤**：价格或净值从阶段高点向下跌了多少。
- **估值百分位**：当前估值在自身一段历史中的相对位置，不代表绝对便宜或未来收益。
- **核心-卫星策略**：核心承担分散和长期回报，卫星承担更主动、更集中的观点表达。
- **价值陷阱**：价格看起来很低，但盈利、现金流或竞争力持续恶化，实际价值下降得更快。
- **风格漂移**：基金实际持仓逐渐偏离原先承诺或历史稳定的投资风格。
- **再平衡**：资产涨跌后把比例调整回预定风险结构，本质是控制风险，不是预测短期市场。
- **幸存者偏差**：只研究最终活下来的优秀产品，忽略已清盘、合并或表现差的产品，导致结论过于乐观。
- **后视偏差**：用今天已知道的赢家去模拟过去选择，结果无法代表当时真实可做的决策。
