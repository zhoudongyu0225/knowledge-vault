## 总体判断

### 1. 微信小游戏进入存量竞争，不等于市场萎缩

证据：

- DataEye：微信 H1 在投产品同比增长 5.52%，明显低于抖音的 54.21%。
- DataEye：微信素材量仍同比增长 12.87%。
- 用户行为：打开次数下降 37.25%，使用时长增加 10.28%。
- 腾讯IAP双周报：IAP小游戏消耗环比仍增长 0.95%。

判断：

```text
产品数量扩张放缓
-> 用户集中到更少、更深的产品
-> 素材和长线运营投入继续增长
-> 竞争从上新速度转向单产品经营效率
```

### 2. 抖音仍是供给扩张市场

抖音 H1 在投产品同比增长 54.21%，素材量同比增长 42.98%，IAA 月度在投占比从 74.73% 升至 79.82%。

适合：

- 新玩法、轻量产品和原生短视频表达的快速验证。
- 高频素材迭代和细分类目抢位。

风险：

- 同质供给进入更快。
- 素材和产品生命周期可能更短。
- 在投量增长不代表单位经济性改善。

### 3. 全量产品和TOP500热投样本是两套结构

- 全量在投产品中，休闲、消除、益智超过 70%。
- TOP500 热投产品中，MMORPG、放置和卡牌合计 71%。

因此：

- 轻休闲占全量产品数量的主体。
- 中重度数值产品在 TOP500 热投样本中更集中。
- 报告没有披露 TOP500 的排序和加权方式，不能把该占比直接解释成预算或收入份额。
- 不能用“某品类产品多”推导“某品类更赚钱”。

### 4. AI已经进入生产和分发，但没有证明利润提升

- DataEye：视频素材中 AI 创作约占 46%。
- 腾讯IAP：AI开始用于视频号达人匹配和选号。

正确定位：

```text
AI负责扩大候选、产能和测试速度
人负责题材、审美、用户匹配和最终判断
项目数据负责证明ROI
```

### 5. 互动广告先改善理解和注册，不自动改善价值

- DataEye：直玩类素材只占素材量 1%。
- 腾讯IAP：试玩消耗占 IAP 小游戏样本 6.20%。
- 腾讯IAA：直玩/试玩 CTR 很高，但成熟 D7 ROI 低于整体。

结论：

互动广告是值得测试的形态，不是已经被证明的高回收方案。评估终点必须是 D7 ROI、留存和LTV。

### 6. 自动化投放会优化目标，不会替人选择正确目标

腾讯IAP聚合样本中：

| 指标 | 非智投 | 智投 | 变化 |
|---|---:|---:|---:|
| 注册成本 | 3.43元 | 2.93元 | -14.58% |
| 首日付费成本 | 1,332.99元 | 2,829.11元 | +112.24% |
| D7付费ROI | 0.130 | 0.084 | -35.38% |

智投解决了便宜注册，但没有在这批聚合数据中解决高价值付费。必须以配对实验排除品类、任务和流量差异。

### 7. 平台政策会制造短期异常

2026-07-15 至 07-31，朋友圈智投九图存在精排 eCPM 加权。晚 7 月九图表现必须标为“政策期”，8 月 1 日后复测，不能直接沉淀为常态素材规律。

## 不能直接横向比较的指标

| 看起来相似 | 实际差异 |
|---|---|
| DataEye纯IAA占比 vs 腾讯IAP小游戏占比 | 前者是跨媒体消耗估算，后者是IAP游戏样本内小游戏与APP的结构 |
| DataEye素材量 vs 腾讯试玩消耗占比 | 一个是素材数量，一个是广告消耗，分母不同 |
| IAA广告变现ROI vs IAP付费ROI | 收入来源和回收曲线不同 |
| H1品类趋势 vs 7月双周环比 | 一个是半年结构，一个是短期波动 |
| 在投产品数 vs 素材榜 | 一个看供给，一个看素材生产强度 |

## 决策调用

| 问题 | 优先调用 |
|---|---|
| 微信和抖音哪个更适合当前产品 | DataEye平台结构 + 项目自身变现模式和素材产能 |
| 某个小游戏品类是否变热 | DataEye在投数、素材量、头部榜单；再补真实ROI |
| 智投是否应该扩大 | IAP双周报 + 同项目智投/非智投配对数据 |
| 试玩/直玩是否值得做 | IAP试玩占比 + IAA成熟ROI + 制作成本 |
| 微信IAA流量位怎么组合 | IAA W2流量位和成熟回收 |
| 晚7月朋友圈九图是否变好 | 先剔除7月15-31日政策加权 |
| ROK/COD能借鉴什么 | 只借鉴素材结构、轻玩法入口和实验方法，不套数值 |

## 最小验证清单

面对一个新的微信小游戏方向，至少补齐：

```text
变现模式：IAA、IAP还是混变
平台：微信、抖音或双端
目标用户与核心玩法
当前CPI/注册成本
D1/D7留存
付费率或广告ARPU
D1/D3/D7 ROI
素材周产能与衰减速度
是否处于平台激励期
```

缺少这些数据时，行业报告只能用于提出测试优先级，不能用于加预算。

## 维护规则

- 腾讯 IAA、IAP 周报按日期独立归档，不覆盖旧文件。
- 半年、年度市场报告更新长期结构；周报只更新近期平台信号。
- 新数据与旧判断冲突时，保留时间、口径和冲突原因。
- 平台激励、节假日和归因回流不完整必须单独标记。
- 没有 ROI、留存或 LTV 支撑的品类热度，只进入“待验证机会”。
