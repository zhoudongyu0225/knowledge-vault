# 老侯说投放：跨行业效果广告方法

## 定位

这是一个**跨行业补充源**，不并入游戏行业事实源。它更适合校验效果广告的通用纪律：先定业务目标、用小预算验证假设、联看前后端转化、管理素材疲劳和建立用户信任。

对游戏投放的使用原则：借方法，不抄阈值；借问题，不把电商案例当游戏 Benchmark。

## 可迁移到游戏投放的五条方法

### 1. 目标从广告指标落到业务结果

文章强调曝光不是大多数效果广告的终点。映射到游戏，至少要把链路写完整：

```text
曝光 -> 点击 -> 安装/进入小游戏 -> 完成核心动作 -> D1/D7 留存 -> 付费/广告收入 -> 回收
```

CTR、CPI 好看但核心动作完成率或留存差，说明素材承诺、人群或产品承接存在断点，不能只在投放后台继续调参。

### 2. 用结构化小测试代替“玄学调账户”

作者给出“2 组创意 × 2 类人群 × 2 种出价”的测试示例。可迁移的不是固定的 `2×2×2` 数量，而是以下纪律：

- 每轮只改变少数可解释变量。
- 给学习期和数据积累留出时间，避免频繁修改导致无法归因。
- 测试前写清假设、主指标、护栏指标和停止条件。
- 成功组合扩大前，再验证它是否能跨时间、地区和素材变体复现。

作者提到的冷启动时长、点击衰减幅度等数字属于经验口径，不作为抖音、快手或海外游戏广告的固定规则；实际执行要看当期平台文档与账户数据。

### 3. 前端转化必须和后端质量接通

电商要看线索是否成交，游戏要看买来的用户是否进入核心循环并留下。建议按素材、人群和计划回传：

- 首局/首关/关键教学完成率。
- 有效游戏时长与关键功能触达。
- D1/D3/D7 留存。
- IAA 展示价值、IAP 付费与预测 LTV。
- 素材承诺和真实玩法的一致性。

这条与 [[2026-05-01-买量三论-投放自动化运营停滞与时间差套利]] 的“平台短期模型与真实长期价值之间存在时间差”可以合并使用。

### 4. 预算要区分放量、探索和机动

作者提出 `80% 成熟素材 + 10% 新方向 + 10% 机动` 的启发式分配。对游戏项目不直接照抄比例，但可以保留三层预算桶：

| 预算桶 | 目的 | 约束 |
|---|---|---|
| 已验证放量 | 扩大已满足回收和质量要求的组合 | 防止素材疲劳、边际成本恶化 |
| 探索测试 | 持续购买新素材、新人群和新卖点的认知 | 单次亏损上限明确，变量可归因 |
| 机动预算 | 应对热点、异常和短期机会 | 不挪用长期回收安全垫 |

具体比例应由项目成熟度、回收周期、素材产能和风险承受能力决定。

### 5. 素材不只抢注意力，还要建立记忆与信任

老侯对直播和视频号人设的拆解可抽象为：受众需求、可证明的身份/优势、稳定记忆点、场景与产品一致、持续兑现承诺。

映射到游戏创意：

- 用稳定角色、视觉符号、口头禅或冲突结构积累系列识别。
- 让题材和场景解释玩法，而不是只做吸睛外壳。
- 每条素材都给用户一个相信承诺的理由，并在进入产品后尽快兑现。
- 爆款模板只能当结构参考，要通过情绪、反转、本地化和产品差异重新表达。

这与 [[游戏本稚-休闲玩法融合观察索引]] 的共享 IP/材质语言、[[杰克Ultra-游戏Marketing-发行创意文章索引]] 的素材分类与验收标准可以互相补足。

## 不应直接迁移的内容

- 电商曝光、留资和成交数据不能替代游戏留存、付费与广告变现数据。
- 直播间主播人设不能等同于游戏世界观或角色 IP；只能借鉴“记忆—信任—兑现”结构。
- 固定测试矩阵、学习期、素材衰减和预算比例都不是平台通用定律。
- 作者举例和数字未做独立数据核验，只能形成测试假设。
