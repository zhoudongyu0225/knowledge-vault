# 个人知识库

这是个人工作和学习知识库的主库，建议用 Obsidian 打开本目录。

## 使用原则

- 看到有用内容，先放进 `00_收件箱`。
- 整理后再移动到 `10_工作`、`20_学习`、`30_项目` 或 `40_资料库`。
- 准备发布到飞书的内容放进 `50_输出`。
- 附件统一放在 `90_附件`，并在笔记中引用。
- 默认不发布到飞书，只有 `publish_to` 显式标记后才发布。
- 分类时以游戏发行和广告相关内容为主轴，其它领域和个人收藏单独标记。

## 目录说明

- `00_收件箱`：主库内部待整理资料。
- `10_工作`：工作项目、会议、团队协作资料。
- `20_学习`：课程、文章、方法学习。
- `30_项目`：围绕具体项目沉淀的资料。
- `40_资料库`：长期可复用的方法论、案例、参考资料。
- `50_输出`：准备同步或发布到飞书的精选内容。
- `90_附件`：图片、PDF、截图等附件。
- `_模板`：常用笔记模板。
- `_索引`：主题首页和导航。

## 分类优先级

1. 游戏发行与广告
2. 其它领域
3. 个人收藏

详细规则见 `_索引/分类规则.md`。
